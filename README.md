# winterhomework
答题网站
答题网站，三个部分。js里面主要是7个函数，ajax请求题目，localStroage存储及排序，开始结束按钮事件，排行榜点击事件，重新答题，计分函数。
css里面就是对页面的一些渲染，有部分CSS3、
html里面主体就是题目骨架以及部分必要的盒子。
